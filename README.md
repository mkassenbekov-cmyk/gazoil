# GAZOIL CRM Prototype

Интерактивный frontend-прототип CRM для коммерческого отдела GAZOIL.

## Что внутри

- Дашборд руководителя отдела продаж.
- Сводка месяца: выручка, литры, сделки, заявки.
- Продажи на сегодня по видам топлива: АИ-92, АИ-95, АИ-98, ДТ лето, ДТ зима, Автогаз.
- Клиенты, обращения, договоры, заказы, продления, возвраты, госзакупки.
- Карточки процессов со стадиями, задачами, документами, комментариями и историей.
- Справочники сотрудников, ролей, видов топлива и юридических лиц.
- Автомаршрутизация:
  - Автогаз -> ТОО «УГХ GAZOIL» -> директор Киікбай А.Б.
  - АИ-92 / АИ-95 / АИ-98 / ДТ лето / ДТ зима -> ТОО «GAZOIL TRADE» -> директор Хуснутдинов Р. Р.
- Сохранение демо-данных в `localStorage`.

## Как запустить локально

Вариант 1: открыть `index.html` в браузере.

Вариант 2: запустить локальный сервер:

```bash
python -m http.server 5173 --bind 127.0.0.1
```

После этого открыть:

```text
http://127.0.0.1:5173/
```

## Структура

```text
index.html   Основная HTML-структура интерфейса
styles.css   Дизайн, адаптивность, Apple-inspired стиль
app.js       Демо-логика CRM, данные, фильтры, карточки, отчеты
```

## Важно

Это пока frontend-прототип. Для полноценной рабочей CRM дальше нужны:

- backend/API;
- база данных;
- авторизация и роли;
- загрузка и хранение документов на сервере;
- интеграция с 1С;
- интеграции WhatsApp, email, телефонии;
- серверный деплой.
